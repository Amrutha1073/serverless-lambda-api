# Serverless Lambda API

🚀 A serverless web application using AWS Lambda and API Gateway.

## 👨‍💻 Technologies Used
- AWS Lambda
- API Gateway
- Node.js (18.x)
- Postman (for testing)

## 🛠 Functionality
- Accepts query or body input and returns a greeting message.

## 📂 Lambda Function Code
Check `index.js` in this repo.
